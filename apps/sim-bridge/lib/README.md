# lib/

Place `Microsoft.FlightSimulator.SimConnect.dll` here.

This file is shipped with the MSFS SDK and is licensed by Microsoft. It is
**not** committed to the repo — every developer supplies their own copy from
the local SDK install.

Standard locations:

- MSFS 2020: `C:\MSFS SDK\SimConnect SDK\lib\managed\Microsoft.FlightSimulator.SimConnect.dll`
- MSFS 2024: `C:\MSFS 2024 SDK\SimConnect SDK\lib\managed\Microsoft.FlightSimulator.SimConnect.dll`

If those paths don't exist, install the SDK from MSFS:
**Developer Mode → Tools → SDK Installer → SimConnect SDK**.
